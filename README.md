# Atmos

App de previsão do tempo com interface moderna, suporte a tema claro/escuro e salvamento de cidades.

# Tecnologias

- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Syne + DM Sans)
- LocalStorage para persistência

# Como rodar

Abra o arquivo `index.html` diretamente no navegador. O projeto não possui dependências externas ou etapa de build.

# Estrutura

```text
atmos/
├── index.html       # Estrutura da página
├── css/
│   └── style.css    # Estilos globais e temas
└── js/
    ├── data.js      # Dados estáticos
    └── app.js       # Lógica da aplicação
